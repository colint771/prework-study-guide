HTML

  • HTML: Hyper Text Markup Language
  • HTML is used to create the structure of a webpage.
  • An Element is a type of content on a webpage such as text, images, headers, and list
  • Elements are also used to divide the content on the page and describe the content to browsers and other developers
  • HTML elements are usually composed of opening tags and closing tags
  • HTML Attributes define additional information about an element and are always defines in the beginning element
  • Metadata is in the head element and is data about the webpage that the browser uses to effectively serve our webpage and users
  • Child Elements are elements that are nested inside of another element
  • Parent Elements are elements that hold all of the child elements
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user

Git

  • Git is a software that tracks changes made to your code
  • GitHub is a cloud-based service that allows developers to store, manage, and collaborate on code
  • Git commands always start with git and are used to save and track versions of our code
  • Repositories AKA Repos are folders used by developers to store files that contain code, files that contain information about the code, and the version history for all of the files in the repo
  • Blockers are a way of saying you're stuck on a problem that is preventing you from moving forward
  • Bottleneck refers to a point in the development process that is slowing down the rest of production
  • GitHub Issues are a task management tool that gives developers and teams an easy way to keep track of key tasks and progress
